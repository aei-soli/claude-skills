# Changelog

## ai-portfolio-manager

### 1.3.0 — 2026-07-15
First public release (SprintFleet Method branding).
- Two-layer method: persistent master tracker (portfolio) + weekly sprint plans.
- Four optimization modes: quality-first, balanced, budget-first, deadline-first.
- Model routing by task class across any mix of AI subscriptions and local compute.
- Usage-budget cycles: verification, burn-down before resets, degradation paths.
- Human-first scheduling: one starred must-do per day, reading days vs decision day.
- Monitoring: daily 2-minute check-in, mid-sprint review, remediation playbook.
- Sprint-end report with per-lane resource usage and carry-over.
- Correlated file system: tracker, roadmap sheets with AI-prompt column, per-project memory files with session logs, numbered project folders.
